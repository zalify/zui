# @zalify-ui/react

Beautifully designed React components built with [Ark UI](https://ark-ui.com) and [Panda CSS](https://panda-css.com).

## Installation

```bash
npm i @zalify-ui/react
```

### Peer Dependencies

- `react` >= 18.0.0
- `react-dom` >= 18.0.0

## Quick Start

```tsx
import { Button } from '@zalify-ui/react'

export default function App() {
  return <Button>Click me</Button>
}
```

Visit [ui.zalify.com](https://ui.zalify.com) for full documentation, component examples, and theming guides.
