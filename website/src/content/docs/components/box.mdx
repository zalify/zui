## Usage

```tsx
import { Box } from '@zalify-ui/react'
```

```tsx
<Box bg="gray.subtle.bg" p="4" borderRadius="l2">
  Content
</Box>
```

Box accepts all style props including `p`, `m`, `bg`, `color`, `borderRadius`, and any other CSS property supported by the design system.
