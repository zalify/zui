## Setup Guide

<Steps>
<Step title="Install the package" number="1">

Install `@zalify-ui/react` and its peer dependency [React](https://react.dev).

```bash multi
npm install @zalify-ui/react
```

</Step>
<Step title="Add the Provider" number="2">

Wrap your application with `ZalifyProvider`. This injects the component styles and sets up theming.

```tsx
import { ZalifyProvider } from '@zalify-ui/react'

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <ZalifyProvider accentColor="jade" grayColor="neutral" borderRadius="md">
          {children}
        </ZalifyProvider>
      </body>
    </html>
  )
}
```

See [Theming](/docs/theming) for all available accent colors, gray palettes, and border radius options.

</Step>
<Step title="Use components" number="3">

Import and use any component from the package.

<ComponentExample component="button" name="basic" codeOnly />

And that's it! Happy hacking!
</Step>
</Steps>
